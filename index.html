Setup API Key
Masukkan Anthropic API key kamu untuk mengaktifkan TechOps KM Assistant. Key ini disimpan di browser kamu saja — tidak dikirim ke mana pun selain Anthropic.
1. Buka console.anthropic.com/settings/api-keys
2. Klik "+ Create key"
3. Copy key yang muncul (sk-ant-api03-...)
4. Paste di bawah ini
Key disimpan di localStorage browser kamu. Tidak ada yang bisa melihatnya selain kamu.
