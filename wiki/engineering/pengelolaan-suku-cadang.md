## Tujuan / Purpose

Panduan pelaksanaan pengelolaan suku cadang agar dapat digunakan sebagaimana mestinya dan menjamin stok suku cadang utama selalu tersedia.

## Ruang Lingkup

Mencakup gudang suku cadang di Departemen Engineering PT EBI.

## Tanggung Jawab

| Peran | Tanggung Jawab |
|---|---|
| Petugas/Administrasi Suku Cadang | Pelaksanaan SOP, penyimpanan suku cadang, buat/kelola Part Code & Database Inventory |
| Teknisi Engineering | Masukan untuk penambahan item suku cadang |
| Supervisor Engineering | Kontrol dan pastikan pelaksanaan SOP |
| Manager Engineering | Evaluasi pelaksanaan; tentukan klasifikasi Critical vs Non-Critical |

## Definisi

| Istilah | Deskripsi |
|---|---|
| Suku Cadang Critical | Dampak langsung ke kualitas produk; lead time pengiriman panjang |
| Suku Cadang Non-Critical | Tidak berkaitan langsung ke kualitas produk; bisa digunakan untuk lebih dari satu mesin |
| Part Code | Kode identifikasi unik per suku cadang; dibuat oleh administrasi; masuk Database Inventory |

## Petunjuk Umum

- Gunakan APD saat memindahkan suku cadang ke tempat lebih tinggi atau pakai tangga
- Penggantian spare part harus sesuai spesifikasi dan fungsi
- Setiap penggantian spesifikasi spare part kritikal harus membuat change control
- Hanya personel berkepentingan yang boleh berada di ruang spare part

## Prosedur

### Identifikasi Suku Cadang
- Suku cadang dibagi dua: Non-Critical (General Part) dan Critical Part
- Setiap suku cadang diberi Part Code untuk memudahkan penanganan dan pengenalan
- Part Code dibuat administrasi dan dimasukkan ke Database Inventory

### Sistem Kode Lokasi
- Ruang penyimpanan: `R. part – R1 . A/B/C` (dari versi PDF sumber)
- Pemantauan inventori: jadwal Senin/Rabu/Jumat
